
# COVID-19 India Data Analysis

Analysis of COVID-19 data for Indian states using Python (Pandas, Matplotlib).

## Project Overview
* Analyzed COVID-19 cases, recoveries, and deaths for various Indian states.
* Visualized data using bar plots and summary statistics.
* Notebook includes data loading, cleaning, and plotting steps.

## Tools Used
* Python (Pandas, Matplotlib)
* Google Colab

## View Notebook
* [covid_india_analysis.ipynb](covid_india_analysis.ipynb)

## How to Run
1. Open the notebook in Google Colab or Jupyter.
2. Run cells to load data and generate plots.

## Dataset
* COVID-19 India data (states.csv or similar)
